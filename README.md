# Riverpod v2 Sample

- Riverpod v2 を使った無限スクロールのサンプル

## Environment

- Flutter v3.3.7
- Dart 2.18.4

## About App

- GitHub リポジトリを検索するアプリ
- Pull-to-Refresh
- 最下部までスクロールすると、次ページのロードを開始
- 追加コンテンツのロード時にエラーが発生したら、最下部にエラーメッセージを表示
